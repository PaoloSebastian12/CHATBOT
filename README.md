# 🤖 WhatsApp AI Chatbot para Empresas

Este es un chatbot inteligente diseñado para automatizar la atención al cliente en empresas. Utiliza IA para responder dudas y está integrado directamente con la API oficial de WhatsApp.

## 🚀 Demo en Vivo
Puedes probar el chatbot enviando un mensaje por WhatsApp al:
👉 **[+1 555-150-3160](https://wa.me/15551503160)**

## 🛠️ Tecnologías
* **FastAPI**: Backend de alta performance.
* **Google Gemini AI**: Cerebro del chatbot.
* **WhatsApp Cloud API**: Integración de mensajería.
* **Google Sheets API**: Para gestión de datos de empresas.
* **Railway**: Hosting y despliegue.

## ⚙️ Instalación Local
1. Clona el repositorio.
2. Crea un entorno virtual e instala las dependencias:
   ```bash
   pip install -r requirements.txt

3. Configura tu archivo .env con las credenciales necesarias.

4. Ejecuta el servidor:

    Bash
    uvicorn main:app --reload
